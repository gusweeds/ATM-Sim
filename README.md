# ATM-Sim

•	Developed an ATM simulation that manages multiple customer accounts with PIN-based authentication. 

•	Implemented object-oriented design principles by creating reusable classes to model customer accounts and banking operations.

•	Utilized ArrayList to store and manage customer account information. 

•	Designed interactive menu functionality supporting deposits, withdrawals, and balance inquiries. 

•	Implemented input validation and error handling to ensure accurate input and program stability.
